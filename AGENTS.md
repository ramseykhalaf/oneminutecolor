# Agents

## Project

iOS companion app for the "One Minute Color" Apple Shortcut. Helps users install the shortcut, run it, and add it to Control Centre. SwiftUI, single screen, includes a ControlWidget extension.

## Commits

Use conventional commits.

## Scripts

- `start_simulator.sh` — build and run on iOS Simulator
- `start_physical.sh` — build, install, and launch on a connected device
- `release_testflight.sh` — archive, export, and upload to TestFlight (reads `.env` for API credentials)
